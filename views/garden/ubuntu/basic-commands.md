### Create Directory
```bash
mkdir testdirectory
```
```bash
mkdir -p /testfolder/directory
```

### Move Directory
```bash
sudo mv /testdirectory /testfolder
```

### List directory content 
```bash
ls -l /home
```
```bash
ls -la /etc/skel # also lists hidden files
```
